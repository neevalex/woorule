== Installation ==

Upload to your plugins/ directory and activate it from wp-admin. Go to WooCommerce -> Settings -> 
Integrations -> RuleMailer and submit an API key and URL (which you get from RuleMailer). 
A new tab should appear in the WooCommerce Settings (if not, try refresh the page).
